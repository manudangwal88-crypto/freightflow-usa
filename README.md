# FreightFlow USA 🚛

FreightFlow USA is a logistics and trucking management web application designed for companies in the United States.

---

## 📌 Features
- Shipment booking system
- Shipment tracking
- Driver and fleet management
- Logistics service pages
- Contact system

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (coming soon)

---

## 📂 Project Structure
- index.html → Homepage
- about.html → About company
- services.html → Services offered
- contact.html → Contact page
- style.css → Styling

---

## 🚀 Future Improvements
- Login system (Admin & Customer)
- Real-time tracking system
- Backend integration (Node.js / PHP)
- Database (MongoDB / MySQL)
- Payment integration
